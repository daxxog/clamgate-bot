# ClamgateBot

  just-dice bot that calls the clamgate.com api

  [![Build Status][travis-image]][travis-url]

Install
-------
stable
```bash
npm install -g clamgate-bot
```
edge
```bash
npm install -g https://github.com/daxxog/clamgate-bot/tarball/master
```
